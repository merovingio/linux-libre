68. O programa IRPF2007 permite a importação de dados do Ganhos de Capital 2006 por meio do arquivo gravado de cópia de segurança?

De forma alguma, pois a cópia de segurança grava dados do Demonstrativo de Ganhos de Capital com ou sem erros de preenchimento.

Para permitir a posterior importação, o programa Ganhos de Capital 2006 deverá ser gravado na opção "Exportar para o IRPF2007", que só é efetuada quando não há erros no preenchimento.
