108. Preenchi a declaração no modelo completo, mas não sei se o simplificado é mais vantajoso para mim. O programa informa a melhor opção?

Perfeitamente. Ao gravar a declaração para entrega à SRF, o programa informa o modelo mais vantajoso para o contribuinte, completo ou simplificado, desde que a declaração tenha sido preenchida no modelo completo, exceto se houver imposto pago no exterior. Se preenchida no simplificado, o programa não dispõe de informações para apontar a melhor opção.
