104. Gravei uma cópia de segurança sem problemas, mas ao restaurar a cópia, apareceu a mensagem "Número de controle inválido no registro NN". O que aconteceu?

É provável que, no início de um campo da declaração, haja caracter especial tal como ", ' @. Exclua o caracter especial.

Se a mensagem ocorrer depois de exportar o arquivo do Ganho de Capital 2006 para o IRPF 2007, verifique se no preenchimento do demonstrativo do Ganho de Capital foi digitado um caracter especial e após a correção, grave para exportar novamente para o IRPF2007. No programa IRPF2007 importe de novo os dados de Ganhos de Capital. A restauração não mostrará mais a mensagem de erro.
