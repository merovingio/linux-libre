98. É possível imprimir o ajuda?

Sim. Na barra de menu Ajuda, selecione a opção "Imprimir Ajuda".
É possível imprimi-lo por tópicos: no alto da página do ajuda existe uma tecla de comando IMPRIMIR (ou PRINT se o seu sistema operacional for em Inglês) que lhe permite imprimir aquela página.
