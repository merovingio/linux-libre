45. No meu comprovante de rendimentos tem informações complementares (item 6). Posso utilizar a opção "Preencher via assistente Comprovante de Rendimentos"?

Sim, mas atente para os valores informados neste item do comprovante de rendimentos e, se for o caso, digite-os nos campos próprios das respectivas fichas. Se, neste campo do comprovante de rendimentos, constar valor, verifique se o mesmo é passível de ser considerado dedução. Se for declaração completa, informe o valor e outras informações na ficha Pagamentos e Doações Efetuados.
