101. O que ocorre se eu restaurar uma cópia de segurança de uma declaração já existente na base de dados?

Se existe uma declaração de ajuste anual do IRPF2007 na base de dados e para o mesmo CPF for feita restauração de cópia de segurança, os dados da restauração se sobrepõem e todos os dados da declaração anterior são substituídos. Não se preocupe, porém, pois o programa alerta ao usuário da existência de uma declaração com o mesmo CPF.
