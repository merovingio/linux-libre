Assistente do Comprovante de Rendimentos
Quadro 1: Beneficiário dos Rendimentos

Veja também...

Os dados dos comprovantes de rendimentos podem ser informados diretamente nas respectivas fichas ou por meio dos quadros desse Assistente. Se utilizar esta última opção, os valores serão digitados nos quadros e transportados para as fichas pelo próprio programa.

Indique o beneficiário do rendimento: titular da declaração ou dependente. Se dependente, informe o CPF do mesmo.

Atenção:
O preenchimento do campo CPF do dependente é obrigatório. Se o dependente estiver dispensado de inscrição no CPF, informe o do titular.

 

Atenção:
Se, após utilizar a opção "Preencher via assistente Comprovante de Rendimentos", for detectada necessidade de retificação, devem ser adotados os seguintes procedimentos:

1) Identificar os dados transportados pelo assistente para as fichas da Declaração.
Na Completa são as fichas Rendimentos Tributáveis Recebidos de PJ, Rendimentos Isentos e Não-Tributáveis, Rendimentos Sujeitos à Tributação Exclusiva/Definitiva e Pagamentos e Doações Efetuados.
Na Declaração Simplificada são as fichas Rendimentos Tributáveis Recebidos de PJ e Demais Rendimentos e Imposto Pago. Para saber as linhas para as quais os dados são transportados, consulte o ajuda dos próximos quadros do assistente.
2) Apagar os dados transportados.
3) Digitar novamente os dados do Comprovante de Rendimentos por meio do assistente ou diretamente nas fichas.
