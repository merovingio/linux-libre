50. O que significam os botões de Consulta/Atualização que aparecem na parte superior da tela em algumas fichas como Bens e Direitos?

Os botões têm os seguintes significados:

Botão utilizado para consultar o primeiro quadro relacionado (rendimento, dependente, pagamento, bem ou dívida).

Botão utilizado para consultar o quadro anterior.

Botão utilizado para consultar o próximo quadro.

Botão utilizado para consultar o último quadro relacionado.

Botão utilizado para abrir novo quadro.

Botão utilizado para excluir um quadro relacionado.
