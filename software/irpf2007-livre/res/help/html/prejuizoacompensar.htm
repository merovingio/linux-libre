Se o valor da linha de Resultado Tributável (linha 07, se atividade rural exercida no Brasil ou linha 06 se atividade rural exercida no exterior) for negativo, o programa transporta esse valor para a linha de Prejuízo a Compensar (linha 08, se atividade rural exercida no Brasil, ou linha 07, se atividade rural exercida no exterior).

Esse valor pode ser compensado com o resultado positivo dos anos-calendário seguintes.
